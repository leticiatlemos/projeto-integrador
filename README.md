# 📚 Projeto: App de Estudos para Foco Neurodivergente

## 📌 Apresentação do Projeto

Este projeto consiste no desenvolvimento de um aplicativo de estudos voltado para estimular o foco de pessoas neurodivergentes, especialmente aquelas com TDAH e outras condições que impactam a concentração e a organização.

A proposta surge a partir das dificuldades enfrentadas por esse público em ambientes tradicionais de ensino e trabalho, onde métodos convencionais muitas vezes não atendem às suas necessidades específicas. O aplicativo busca reduzir essas desigualdades por meio de uma abordagem personalizada, interativa e adaptada, tornando o processo de aprendizagem mais acessível, eficiente e engajador.

---

## 🎯 Objetivos do Projeto

Os principais objetivos da solução são:

- Auxiliar pessoas neurodivergentes a desenvolverem maior foco nos estudos;
- Oferecer uma plataforma de aprendizado personalizada e adaptável;
- Tornar o processo de estudo mais atrativo e estimulante;
- Reduzir as barreiras enfrentadas por esse público no ambiente educacional;
- Contribuir para o desenvolvimento profissional e pessoal dos usuários.
